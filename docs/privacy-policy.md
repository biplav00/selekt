# Privacy Policy

**Selekt** ("the Extension") is a Chrome extension designed to help developers generate test automation locators.

## Data Collection

The Extension does **not** collect, store, or transmit any personal information or browsing data to external servers.

## Local Storage

- **Locator History**: Locators generated during your session are stored locally in your browser's storage. This data remains on your device and is not transmitted anywhere.
- **Settings**: Your preferences (default format, history limit) are stored locally.

## Permissions

The Extension requires the following permissions:
- `activeTab`: To access the current tab for element selection
- `scripting`: To inject scripts for DOM analysis
- `tabs`: To communicate with browser tabs
- `storage`: To persist your settings and locator history
- `sidePanel`: To display the extension UI

These permissions are used solely for the Extension's core functionality of generating CSS/XPath selectors for web elements.

## Third-Party Services

The Extension does not integrate with any third-party analytics, advertising, or tracking services.
